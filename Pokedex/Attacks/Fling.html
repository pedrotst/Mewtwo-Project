Battle Effect:
The user flings its held item at the target to attack. Its power and effects depend on the item.
In-Depth Effect:

As the Base Power can get up to 130, it adds major strategy possibilities, especially on Dark Type Pokémon which get the Same Type Attack Bonus from this move. Causing 195 Base Power maximum

There are several Base Power possibilities for the items. As every item is capable of being thrown, the lists are quite extensive so use this table to navigate to see what item corresponds to the power:
